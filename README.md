# Phishing Website Detector

This is one of my starting projects while learning Python. This phishing website detector detects the vulnerability of the website and categorizes the website URL into safe, suspicious and dangerous by entering a URL. The logic behind this code is simple. It tries to read the URL and detect the use of suspicious characters like @, HTTP instead of HTTPS, length of URL, etc to determine the safety of the website.
